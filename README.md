contableplus
============

Bank account contability

####PLEASE NOTE
If you want to be able to run graphics for any account, you should have tkinter installed.  
Linux: `sudo apt-get install python*3*-tk`  
Mac: `nothing`  
Windows: `Good luck`  

#### How To:
Simply run userInterface.py and that's it!
